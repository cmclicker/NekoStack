# @nekostack/theme — Roadmap

Authoritative source for "what ships when." Full design rationale lives in the package [`README.md`](../README.md).

## v1.0 — Stable token system

Status: **shipped**. Apache-2.0. Published to npm.

Includes:
- Design tokens spec (DTCG format) — color, typography, spacing, radius, shadow, motion, z-index, layout
- CSS custom-property output (`neko.css`) with semantic aliases
- Multi-theme system: synthwave dark, synthwave light, system-default
- Tailwind v3/v4 preset
- Scrollbar tokens and theme variants
- Zero JS runtime — pure CSS

## Future

No additional milestones currently planned. Token additions are additive and ship as patch releases.
