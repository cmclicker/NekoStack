@nekostack/theme

Theme (brand personality) and mode (dark/light) are orthogonal — switch each independently. A consumer site sets <html data-theme="neko" data-mode="dark"> and the whole token set retargets. Adding a new theme or mode is one key in tokens.json + rebuild.
